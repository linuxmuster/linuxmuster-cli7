# up

Check if devices are online (LINBO, Linux, Windows, or unresponsive).

## Usage

```
lmncli up [GROUP] [OPTIONS]
```

| Option | Description |
|---|---|
| `GROUP` (argument, optional) | Restrict the check to a single device group |
| `--school`, `-s` | Target school (default: `default-school`) |

```
lmncli up
lmncli up win10-room1
```
