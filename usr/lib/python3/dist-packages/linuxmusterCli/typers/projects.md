# projects

List sophomorix projects whose hostname, ip, or member contains FILTER_STR.

## Usage

```
lmncli projects [FILTER_STR] [OPTIONS]
```

| Option | Description |
|---|---|
| `FILTER_STR` (argument, optional) | Filter on project name (case-insensitive) |
| `--school`, `-s` | Target school (default: `default-school`) |

```
lmncli projects
lmncli projects robot
```
